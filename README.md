# スターカート (STAR KART)

お菓子の王国「シュガリア」を舞台にした、ブラウザで動く3Dカートレースゲームです。
多様な人外レーサーたちが専用マシンでグランプリを競います。
キャラクター・コース・アイテム・音楽はすべてオリジナル。外部アセット・ビルドツール不要で、
すべてコードからプロシージャル生成されます。

## 起動方法

**`index.html` をダブルクリックしてブラウザで開くだけです**(Chrome / Edge 推奨)。

うまく動かない場合はローカルサーバ経由でも起動できます:

```
python -m http.server 8123
→ ブラウザで http://localhost:8123 を開く
```

※ 音はブラウザの仕様上、最初のキー入力またはクリックの後から鳴り始めます。

## 操作方法

| 操作 | キー |
|------|------|
| アクセル | ↑ または W |
| ブレーキ / バック | ↓ または S |
| ハンドル | ← → または A D |
| ドリフト(ホップ) | Space または Shift |
| アイテム使用 | E または Enter |
| ポーズ | Esc または P |
| メニュー決定 | Enter または Space |

### ドリフト&ミニターボ
走行中に **Space+ハンドル** でドリフト。ドリフトを続けると火花が
**青 → オレンジ → 紫** と変化し、離した瞬間にブーストが発動します(長いほど強力)。

### ロケットスタート
カウントダウンの「2」あたりからアクセルを押しっぱなしにすると、スタートダッシュが決まります。

## ゲームモード

- **グランプリ** — 3コース連戦のポイント制(1位15pt〜)。総合優勝を目指し、表彰式まで楽しめます
- **シングルレース** — 好きなコースで1レース(8台)
- **タイムアタック** — アイテムなしの単独走行。ベストタイムは自動保存されます

## キャラクター(9体)

マカロン・ププル / ドーナ・リング / タプリン / ソフクリン / ソーダ・シュワリ /
ワッフル・グリッド / ショコラ・ノワール / バウム・ロール / ボンボン・キャノン

軽量級(加速・曲がりやすさ重視)〜重量級(最高速・押し合い重視)まで性能が異なります。

## コース(3本)

1. **クッキータウン・ロード**(初級) — 焼きたてクッキーの街並みを走る、広くて走りやすいコース
2. **チョコレート・キャニオン**(中級) — チョコの川に落ちないように。ビスケット筏のジャンプが見どころ
3. **レインボーシュガー・スカイキャッスル**(上級) — 雲の上の飴細工の城。浮遊足場と急降下が待つ最難関

## アイテム(10種)

はちみつブースト / いちごスコーン弾 / キャラメルトラップ / 追尾マシュマロ / クッキーシールド /
サワーレモン爆弾 / スターゼリー(無敵) / ソーダの嵐 / レインボースプリンクル(下位救済) / ミラクルパフェ

順位が低いほど強力なアイテムが出やすくなっています。

## 技術メモ

- Three.js r134(UMD、`lib/three.min.js` に同梱)+ Vanilla JS。ES Modules・ビルドツール不使用
- 3Dモデル/テクスチャはプリミティブ+CanvasTextureで全てコード生成
- BGM・効果音はWeb Audio APIによる全合成(チップチューン風シーケンサ)
- 設計資料は `DESIGN.md`、開発経緯は `PROGRESS.md` を参照
